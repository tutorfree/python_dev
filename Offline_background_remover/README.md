# Removedor de Fundo – Offline 🖼️✂️
<img width="787" height="561" alt="Removedor de Fundo Offline" src="https://github.com/user-attachments/assets/b292c683-1895-4e8e-bef2-30abae4b7d78" />


Aplicativo simples para **remover o fundo de imagens automaticamente**,  
sem precisar de internet, cadastro ou serviços online.

Funciona **100% offline**, diretamente no seu computador.

---

## ✅ O que este programa faz

- Remove o fundo de imagens (PNG, JPG, JPEG)
- Usa Inteligência Artificial local (não envia nada para a internet)
- Funciona sem instalar Python
- Interface simples, basta clicar

---

## 🔒 Privacidade

- Nenhuma imagem é enviada para a internet
- Nenhum dado é coletado
- Tudo acontece **somente no seu computador**

---

## 💻 Requisitos

- Windows 10 ou superior
- Não precisa de internet
- Não precisa instalar nada

---

## ▶️ Como usar

1. Abra o arquivo:


2. Clique em **Selecionar imagem**
3. Aguarde o processamento
4. A imagem com fundo removido será salva automaticamente

---

## 📁 Onde a imagem é salva?

- Na **mesma pasta** da imagem original
- Com o fundo removido (normalmente em PNG)

---

## ⚠️ Observações importantes

- Na primeira execução pode demorar um pouco (é normal)
- O programa é maior que aplicativos comuns porque inclui a IA completa
- Alguns antivírus podem alertar por ser um `.exe` independente  
(isso é comum em programas portáteis)

---

## 🌱 Sobre este projeto

Este programa foi criado com o objetivo de:

- ser acessível
- funcionar sem internet
- ajudar pessoas com poucos recursos
- respeitar a privacidade do usuário

Ele pode ser distribuído livremente.

---

## 🛠️ Tecnologias utilizadas (informativo)

- Python
- Tkinter (interface)
- rembg (remoção de fundo)
- ONNX Runtime (IA local)
